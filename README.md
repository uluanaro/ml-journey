# ML Journey

**A structured, hands-on path toward a Junior Machine Learning / Data Scientist role.**

This repository documents my end-to-end learning journey through classical machine learning — built from the ground up, week by week, with a working artifact at the end of every stage. Each project is written to be read: clear code, explicit reasoning, and a focus on *understanding the mechanics* rather than calling libraries blindly.

---

## Why this repository exists

Instead of scattered notebooks, this is a single, disciplined track with a fixed roadmap, deadlines, and review checkpoints. The goal is not to collect tutorials, but to build a portfolio of small, correct, well-documented projects that demonstrate how I think and work.

## Roadmap

The track is organized into four cycles (~20 weeks), each ending with a practical, interview-ready project:

| Cycle | Focus | Key project |
|-------|-------|-------------|
| **1 — Python for DS** | Python fundamentals, NumPy, Pandas, EDA, SQL | Real-dataset analysis with business conclusions |
| **2 — Classic ML core** | Regression, classification, trees, boosting, metrics, validation | Trained model served via a `/predict` API + demo |
| **3 — Applied ML & production** | Clustering, dimensionality reduction, time series, DL overview | ML pipeline (Airflow + MLflow tracking) |
| **4 — Interview preparation** | ML system design, project framing, resume, mock interviews | Portfolio finalization |

Algorithms (NeetCode) are practiced in parallel throughout the track.
